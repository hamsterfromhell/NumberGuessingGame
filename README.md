  <h3>‚ú® Features</h3>

  | Feature                  | Description                                                                 |
  |--------------------------|-----------------------------------------------------------------------------|
  | 3 Difficulty Levels      | Easy (1-50), Medium (1-100), Hard (1-200) with color-coded cards            |
  | Smart Feedback           | "Too high" / "Too low" hints + color changes                                |
  | Persistent Stats         | Best score, games won, win rate, average attempts (saved between sessions)  |
  | Pro Tips Section         | Learn optimal strategy with binary search hints                             |
  | Modern Tkinter UI        | Rounded buttons, gradients, circular indicators, responsive layout          |
  | Play Again & Reset       | Quick replay + full stats reset option                                      |

  <h3>üéÆ How to Play (Locally on Replit / Your Machine)</h3>

  1. Clone / fork the repo or open in Replit
  2. Install dependencies (if needed): `pip install tk` (usually pre-installed)
  3. Run the main file: `python main.py` (or whatever your entry file is named)
  4. Choose difficulty ‚Üí start guessing!
  5. Use the **Pro Tips** to improve ‚Äî binary search can get you wins in very few tries!
